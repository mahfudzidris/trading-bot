AI Trading Bot — Architecture

TwelveData → Ensemble Strategies → Market Sentiment → DeepSeek AI Meta-Analyzer → Auto-Trade (Bracket TP/SL) → Alpaca Broker → Dashboard · 24/7 Market Watcher Loop · TP/SL Auto-Recalibrate · DB Persist · Error Tracking

Data Flow

  • 📡 TwelveData → Real-time OHLCV + RSI/SMA/EMA/Volume indicators
  • 🧠 3 Ensemble Strategies → Weighted voting → Consensus signals
  • 🤖 DeepSeek AI Meta-Analyzer → BUY/SELL/HOLD with reasoning + TP/SL
  • 🏦 Alpaca Paper → Market orders, bracket TP/SL, positions sync, account tracking
  • 🗄️ SQLite → Persistent storage for trades, backtests, & reports
  • 🔄 Market Watcher → 24/7 background loop, runs every N min during market hours
  • 🛡️ TP/SL Recalc → Auto-adjust stale TP/SL to current market price via get_current_price()
  • 💾 DB Persist → Watcher saves trades to SQLite with recalibrated TP/SL values
  • ⚠️ Error Track → Watcher exposes last_tick_error + failed_ticks in /api/health

Ensemble Strategies

  • 📈 Trend Following — SMA20/SMA50 crossovers, price vs MA, slope (7 rules)
  • 🔄 Mean Reversion — RSI extremes (<30 OVERBOT, >70 OVERBID) + volume confirmation
  • ⚡ Momentum — Price ROC + volume momentum scoring
  • 🔀 Consensus — Weighted voting with per-run random weights, fed into AI prompt
  • 🧪 Mock mode — Ensemble consensus drives decisions when no AI key

Dashboard Pages

  • 📊 Dashboard — PnL chart, AI Card with Buy/Sell, Quick Actions, Positions
  • 🧠 Strategy — Live indicators + AI prompt preview + ensemble signals (new!)
  • 📈 Analysis — Per-symbol AI recommendations with strategy signals card
  • 🔄 Trades — Trade history with filters & pagination
  • 🔬 Backtest — Run historical simulation with full metrics (Sharpe, DD, Win Rate)
  • ⚙️ Settings — API config, mock/auto-trade toggle, Market Watcher 24/7 ON/OFF
